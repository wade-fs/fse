Understanding is the residue of repeated constraint violation attempts.
（理解，是在限制中反覆試錯留下的殘留物）

A world produces understanding when constrained action creates persistent epistemic gaps.
（世界在受限行動中產生持續認知落差時，就會產生理解）

核心不是世界，也不是知識，而是這個循環：
          ┌──────────────────────┐
          │   1. WORLD (constraints) │
          │  世界 = 有限制的結構     │
          └─────────┬────────────┘
                    ↓
          ┌──────────────────────┐
          │  2. INQUIRY (traversal) │
          │ 行動 = 在限制中試探     │
          └─────────┬────────────┘
                    ↓
          ┌──────────────────────┐
          │  3. CONSEQUENCE        │
          │ 世界回應（不可完全預期）│
          └─────────┬────────────┘
                    ↓
          ┌──────────────────────┐
          │  4. GAP / CONFUSION    │
          │ 認知落差（關鍵核心）   │
          └─────────┬────────────┘
                    ↓
          ┌──────────────────────┐
          │  5. ADJUSTMENT         │
          │ 修正下一次行動         │
          └─────────┬────────────┘
                    ↓
                 (loop)
