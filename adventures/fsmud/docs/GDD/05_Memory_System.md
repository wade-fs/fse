# 05. Memory System - 歷史記憶碎片系統 (Memory System)

記憶 (Memory) 是對抗遺忘、推動歷史版本前進的唯一鑰匙。

---

## 🧩 記憶的屬性與解鎖
每個記憶碎片以 YAML 定義在 `/mudlib/data/yaml/memories/`，其結構包括：
*   `trigger_site`：玩家進入哪一個特定地標時會觸發檢測。
*   `conditions`：解鎖此記憶碎片的前置要求（如特定踏印、前置解鎖的記憶 ID、或是職涯等級）。
*   `quality`：記憶的品質與稀有度（`1`=殘片, `2`=記錄, `3`=傳承, `4`=史詩, `5`=源流）。
*   `progress`：解鎖成功後回饋給全服時代進度的權重值。

---

## 🧘 歷史共鳴 (Resonance)
*   **被動共鳴**：同一個地標內有 2 位以上玩家同時在場，且地標未處於冷卻期 (120秒) 時，會引發群體共鳴，回升聚落記憶度並獎勵在場玩家職涯修練點。
*   **主動共鳴儀式 (Commune)**：符合文士 (scholar >= 2) 或農夫 (farmer >= 3) 條件的玩家，可使用 `commune` 指令主動對該地標引導共鳴，回升記憶值。
