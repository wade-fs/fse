# 08. Faction System - 勢力聲望系統 (Faction System)

玩家可以加入不同的在地陣營，爭奪對這片土地的話語權與影響力。

---

## 🏛️ 三大勢力定位
1.  **劉家 (Liu Family)**：代表清領與日治時期的地方豪族、開發墾首，掌握大量地契與農田資源。
2.  **糖業組合 (Sugar Industry)**：代表日治時期引領台灣經濟起飛的現代化糖鐵與製糖產業利益。
3.  **廟委 (Temple Committee)**：代表聚落的民間傳統信仰（如大士爺廟、天后宮），維繫信仰安定、鎮守邪祟與遺忘危機。

---

## 📈 聲望與任務關聯
*   **聲望儲存**：玩家與各勢力的好感度與聲望，在 `/std/user.c` 進行持久化存檔。
*   **獎勵與裝備解鎖**：完成特定勢力的委託（如劉家地契任務）可獲得專屬勢力聲望與特有配備（如劉家家徽戒指、廟委鎮煞護符），這些配備是玩家解鎖更高級歷史層的重要憑證。
