# 03. Era System - 歷史時代推展系統 (Era System)

《源流福爾摩沙》的世界是由全伺服器共用的「當前版本時代 (Current Era)」所驅動的動態世界。

---

## ⏳ 時代推展流程
1.  **事件驅動**：當玩家解鎖任何歷史記憶碎片時，`memory_d` 會發送 `MemoryCompleted` 事件，夾帶記憶點數（Progress）。
2.  **進度累積**：`timeline_d` 監聽該事件並將數值累加至 `world_progress` 全域變數。
3.  **時代跳躍**：當 `world_progress` 達到當前時代 YAML 定義的 `min_progress` 門檻時，自動觸發 `next_era()`：
    *   全服廣播時代躍遷訊息。
    *   發送 `EraShifted` 事件，重繪地理通道，並重置所有聚落記憶值至安全水準（不低於 40%）。

---

## 📜 時代清單與 ID 格式
*   **v0_1 (荒野紀/原住民時期)**：史前至 1624 年。
*   **v0_2 (海商紀/荷西與明鄭時期)**：1624 至 1683 年。
*   **v1_0 (清領前期)**：1683 至 1895 年。
*   **v2_0 (日治/乙未時期)**：1895 年。
*   **v2_1 (日治中後期/大正與西來庵時期)**：大正、昭和年間。
*   **modern (現代)**：當前年代。
