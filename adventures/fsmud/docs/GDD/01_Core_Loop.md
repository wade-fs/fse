# 01. Core Loop - 核心循環系統 (Core Loop)

《源流福爾摩沙》完全揚棄傳統戰鬥升級、爆肝刷裝的數值膨脹機制。其核心體驗是「理解、探索與復興」。

---

## 🔄 核心循環鏈
```text
探索土地（Travel & Look）
      ⬇️
解鎖地理與人文「踏印」 (Footprints)
      ⬇️
引發歷史「共鳴」 (Resonance)，收集記憶碎片
      ⬇️
提升聚落記憶度與文明指標 (Settlement Memory %)
      ⬇️
累積職涯點數 (Career Points)
      ⬇️
推動世界時代演進 (Era Progression)
```

---

## 🔀 基礎指令與世界觀內核的融合

### 1. `look` (查看)
*   **物理層**：顯示 2D Minecraft 風格的方塊地圖與物件。
*   **語意層**：自動結合所屬聚落的當前文明指標（人口、記憶完整度、主要產業），並顯示動態失源者危機狀態。
*   **天賦層**：若玩家擁有「島嶼記憶」天賦且符合前置條件，會顯示扭曲的歷史層入口。

### 2. `travel <目的地>` (旅行)
*   **語意旅行**：藉由道路與交通網絡進行地理移動，取代傳統 `east`、`west` 等瑣碎的方向控制。
*   **時代過濾**：地理網絡會根據當前時代動態啟用或限制部分路線（如日治縱貫鐵路僅在日本時代與現代開放）。

### 3. `explore <地標>` (探索)
*   探索特定 Site 觸發地理與歷史踏印的發放，回饋聚落记忆度。
