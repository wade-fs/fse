# 09. Career System - 四大職涯系統 (Career System)

職涯 (Career) 是玩家與世界互動、獲取資源與理解歷史的專業工具。

---

## 🛠️ 四大職涯與專屬技能指令

### 1. 農人 (Farmer) — 指令：`farm`
*   **職責**：耕作與開墾。在農地中收集本土特產與進行水利疏濬。
*   **特色**：等級高時，可使用 `farm` 解鎖早期先民開墾的歷史碎片。

### 2. 商人 (Merchant) — 指令：`trade`
*   **職責**：商路開闢與交易。在市場與古老港口中調配貨物與交易。
*   **特色**：可透過交易開啟新時代商路（如清領大稻埕、安平五條港）。

### 3. 匠人 (Artisan) — 指令：`craft` (修復：`repair`)
*   **職責**：工藝製作與設備搶修。
*   **特色**：在鹿港等地可雕刻木雕、製作手工香；也是搶修糖鐵火車頭等硬體任務的核心。

### 4. 文人 (Scholar) — 指令：`record` (考據：`research`)
*   **職責**：歷史文獻記錄與共鳴。
*   **特色**：考據碑文、編寫並發表地方風土誌以回升聚落記憶度。
