# 源流福爾摩沙 - 歷史時間線

## 世界觀原則

歷史不是背景。

歷史本身就是遊戲版本。

每一次版本更新，都是世界翻頁。

玩家無法改變歷史結果。

但可以影響：

* 記憶保存度
* 地方認同度
* 踏印深度
* 文化傳承度
