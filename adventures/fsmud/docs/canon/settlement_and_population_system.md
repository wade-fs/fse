# 聚落升級條件

不是人口。

而是綜合發展。

---

## 升級公式

```text
人口
+
產業
+
文化
+
記憶
+
商路
+
凝聚
```

---

任何單項不足。

無法升級。

---

# 特殊規則

## 記憶不可缺失


```text

```

