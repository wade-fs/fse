# 源流福爾摩沙 - 勢力設定

## 設計原則

本作不存在：

* 帝國陣營
* 國家陣營
* 絕對正義陣營
* 絕對邪惡陣營

所有勢力皆來自：

* 土地
* 家族
* 聚落
* 信仰
* 產業
* 記憶

玩家可同時加入多個勢力。

勢力彼此重疊。

形成台灣特有的網絡社會。
