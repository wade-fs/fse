# 《史前文明》生態生存與生活手冊 (PC Living Manual)

這不是一份通關攻略，而是一份在史前地球的「生活指南」。
在 Flow State Engine (FSE) 中，世界是具有物理一致性的不變量。你的目的不是「走出這個遊戲」，而是在三疊紀的燥熱、侏羅紀的潮濕與白堊紀的末日中，利用物理規律永續地生活下去。

---

## 一、 日常生活的生存節奏 (Daily Survival Rhythm)

在 FSE 中，玩家受到體力 (fatigue)、飢餓度 (hunger) 與口渴度 (thirst) 的即時約束。這三個數值構成了你的「熵值」。

```
                 【 體力 (Fatigue) 】
                    /           \
         [行動與摩擦消耗]     [避風岩洞/樹冠木屋休息]
                  /               \
 【 飢餓度 (Hunger) 】             【 口渴度 (Thirst) 】
     /         \                       /          \
[背景流逝]   [吃烤熟蟲肉/恐龍蛋]  [背景流逝]   [飲用沸水/地下泉水]
```

### 1. 體溫與水份調節 (Temperature & Hydration)
*   **三疊紀 (55°C 高熱)**：水份流失速度是正常的 3 倍（`thirst_decay: 3.0`）。避免在烈日下奔跑或待在空地上。你必須以 `triassic_shade` (背光洞窟) 或 `stone_arch` (天然石拱門) 為家，在清晨或傍晚外出採集。
*   **侏羅紀 (極度潮濕)**：水份流失慢，但空氣悶熱，食物腐爛速度極快。採集到的 `bug_meat` 或 `bird_egg` 必須在一天內食用完畢，否則會變質。

### 2. 生火作為生活的核心
火不是開門的道具，火是你的保護傘：
*   **每日例行生火**：每當夜幕降臨（進入 `first_night` 階段的房間），第一件事是生起營火（`light campfire`）。營火不僅能抵禦失溫，還能防止暗處的節肢動物靠近。
*   **利用餘燼**：你點燃的營火會產生 `embers` (乾燥火種)。妥善將火種收在背包中，它是你前往下一個未知區域（如 `dark_cave` 照亮岩壁、`lava_tube` 觀測對流）的移動熱源。

## 二、 跨階段工具的保養與重複利用

你所獲得的每一樣工具都具有「耐久度 (durability)」。工具會損壞，你需要過上「採集、製作、保養」的循環生活。

| 工具 | 獲取階段 | 跨階段日常重用價值 |
|---|---|---|
| **磨尖的骨刀 / 燧石刃** | Novice / First Night | 1. 割取藤蔓纖維製作繩索<br>2. 剝下巨木樹皮條製作防雨庇護所<br>3. 在紅樹林削尖木棒製作魚叉捕魚<br>4. 在獵捕大型昆蟲時進行無傷防衛切割 |
| **新鮮松脂 (pine_resin)** | Bug Eater (三疊紀) | 1. 與木棒綁定，作為長效燃燒的防風火把燃料<br>2. 作為黏合劑，在樹冠層加固即將崩塌的藤蔓吊橋<br>3. 塗抹在簡易木筏裂縫上，進行防水密封 |
| **簡易抓鉤** | Jurassic (侏羅紀) | 1. 不僅能攀爬樹冠，在陡峭的 `fossil_cliff` (化石峭壁) 也能作為攀岩攀爬的安全輔助<br>2. 在遇到急流時，可用於套住對岸岩石，拉動木筏渡河 |
| **泥沙與海藻泥** | Cretaceous (白堊紀) | 1. 塗抹全身阻絕哺乳動物氣味，避免被霸王龍嗅出<br>2. 在地熱噴口附近塗抹雙手，作為短暫的隔熱層 |

## 三、 地球的一致性：時間與空間的交織

這個世界沒有「關卡封閉性」，你正站在百萬年地質歷史的交會點上。

### 1. 化石是過去的眼睛
*   當你在 `wind_canyon` (三疊紀) 撿起一塊半透明的 `mica` (偏振雲母片) 時，那是因為這片土地在千萬年前曾是海洋，雲母是石灰沉積物在壓力下結晶的產物。
*   當你在 `fossil_cliff` (侏羅紀) 看到菊石化石時，請記住，這片峭壁在三疊紀曾是一條波濤洶湧的深谷。

### 2. 你是先祖，也是後代
*   在 `last_stand` 終局中，小行星即將撞擊地球。你在 `cave_gallery` 牆上利用赭石顏料畫下的壁畫（`add painting` / `draw`），在物理上就是三疊紀人在 `dark_cave` 中點燃火把、驚嘆並解讀風向與歷史規律的同一幅壁畫。
*   你在地下保存庫（`vault_entrance`）中封存的知識，會在冰封百萬年後，被下一個甦醒的「穿越者」在廢墟中重新掘出。

這不是一場解謎遊戲。這是你在史前地球上，利用不變的物理規律，所刻下的一段生活史詩。
