# Shushan Odyssey: The Spiritual Reality Manifesto
# 蜀山遊記：精神真實核心宣言

> "Every Adventure asks one question: 'What is true in this world?'"
> —— 每個冒險都在問同一個問題：「在這個世界裡，什麼是真實的？」

在 FSE (Flow State Engine) 的全新認識論架構中，修行（Cultivation）不再是數值與經驗的堆疊，而是對世界因果規律的探索、困惑與領悟。

---

## 核心認識論：四個動詞 (The Four Verbs)

精神世界（Spiritual Reality）乃至於物理與社會世界，其因果運作可以完全用科學的認識論來描述：

1. **Observe (觀察)**
   * 玩家能感知什麼，完全取決於他當前的理解深度。
   * 沒有心氣共鳴時，雷聲只是雷聲；領悟了雷法，同一道雷便 Reveal 出「天道運轉的節奏與天劫開闔的時機」。
   
2. **Predict (預測 / 行動即預測)**
   * 玩家的行動（Action）本質上就是他的假說與預測。
   * 當玩家發出 `meditate` 指令，他隱含的預測是：「我此時的狀態足以與天地靈氣共鳴」。世界不要求玩家用文字解釋他的理解，他的行動已經揭示了它。

3. **Evaluate (評估)**
   * 系統將玩家隱含的預測（行動與攜帶的 Evidence），比對 Reality 庫中的客觀物理/精神法則。
   * 評估出玩家當前的 **Understanding State**。

4. **Evolve (演化)**
   * 世界根據評估結果進行動態演化。
   * Evolve 不是遊戲對玩家的「對錯懲罰」，而是 Reality 對玩家錯誤認知模型的**物理回饋修正**。Evolve 會產生新的現象（Signals / Observations），將玩家帶回下一個 `Observe` 循環。

---

## 認知狀態的三個層次 (Understanding States)

評估不應該只有二元的「成功」或「失敗」，而是玩家的認知模型與世界真實規律的對齊程度：

* **Understanding (理解 / 對齊)**
  * 玩家的認知模型與 Reality 規律完全對齊。世界依其預期回應，解鎖更深的感知。

* **Misunderstanding (偏誤 / 局部偏離)**
  * 玩家的認知模型有盲點。例如：知道要迎風引火，但忽略了空氣溼度。這會帶來局部的意外後果（Consequence），引導玩家發現缺失的 Evidence。

* **Misconception (迷思 / 主動錯誤)**
  * 玩家帶著主動錯誤的「成見」行動。例如：以為越強求打坐、修為加得越快。這個錯誤信念無法用溫和引導糾正，需要被 Reality 撞碎，觸發重度 **Confusion (困惑)**。

---

## 閉合的 Discovery Loop

這四個動詞構成了 FSE 的心跳循環：

```
      Reality.Observe (感知取決於理解)
            │
            ▼
      Player Action (= Prediction 預測)
            │
            ▼
      Reality.Evaluate (比對 Laws 評估)
            │
            ▼
      Reveal / Memory / Discovery (認知更新)
            │
            ▼
      Reality.Evolve (世界演化，產生新 Observations 訊號)
            │
            └────────── 餵回給下一個 Observe 循環
```
